# HotKey-Tracker

HotKey-Tracker is a simple C++ program designed to help users keep track of keyboard shortcuts (hotkeys) across different applications and systems.

Many programs, tools, and operating systems rely heavily on shortcuts to improve productivity. However, remembering dozens of hotkeys across multiple platforms can quickly become overwhelming.

This program aims to solve that problem by providing a centralized way to store, organize, and manage your frequently used shortcuts.

## Features
- Store commonly used hotkeys
- Organize shortcuts by application or category
- Quickly reference saved shortcuts
- Lightweight and easy to use

## Why HotKey-Tracker?
Whether you're a developer, designer, gamer, or power user, remembering every shortcut for every application can be difficult. HotKey-Tracker helps you stay productive by keeping all your important shortcuts in one place.

## Future Improvements
- Search functionality for finding shortcuts faster
- GUI support
- Import/export shortcut lists
- Custom shortcut categories
