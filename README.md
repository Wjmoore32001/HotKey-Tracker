# HotKey-Tracker

HotKey-Tracker is a **CLI-based C++ application** designed to help users keep track of keyboard shortcuts (hotkeys) across different applications and systems.

Many programs, tools, and operating systems rely heavily on shortcuts to improve productivity. However, remembering dozens of hotkeys across multiple platforms can quickly become overwhelming.

This project aims to solve that problem by creating a simple command-line tool where users can store, organize, and manage their frequently used shortcuts in one place.

---

## Why I Built This
I often found myself forgetting shortcuts across different applications and constantly searching them up again. I wanted to build something that solves that issue while also giving myself an opportunity to improve my C++ skills through a practical project.
